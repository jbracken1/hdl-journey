# hdl-journey

A collection of HDL projects and exercises as I learn FPGA and digital hardware engineering.

I'm Jack, an ECE student at Virginia Tech interested in FPGA design, RTL engineering, 
and DSP. This repo documents my progress learning Verilog and working toward a career 
in digital hardware.

## Structure
- `verilog/` — combinational logic, sequential logic, and FSM exercises
- `fpga-projects/` — designs deployed to the Basys 3 board
- `capstone/` — end of summer project (ALU, UART, or FIR filter)

## Tools
- Intel Quartus Prime
- Basys 3 FPGA board (Digilent)
- Icarus Verilog + GTKWave for simulation
